# DashHR MVP

Minimal  MVP built with React + Vite + TypeScript + Tailwind.

To run locally:

# For the front end:
- on a bew terminal

```bash
# install
npm install

# dev
npm run dev
```
- on a new terminal:

```bash
cd Backend
pip install -r requirements.txt
uvicorn main:app --reload
```

Notes:
- Uses localStorage for mock application storage.
- No backend; placeholder data in `src/data/mock.ts`.
